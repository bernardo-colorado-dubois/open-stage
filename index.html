Cómo funciona

Simple. Declarativo. Componible.

Construye pipelines conectando tres tipos de componentes con pipes. Los datos fluyen de izquierda a derecha.

01 / ORIGIN

Produce datos

Lee un CSV, consulta una base de datos, llama una API REST. Un Origin no tiene entradas y genera exactamente una salida.

02 / NODE

Transforma datos

Filtra, agrega, une, copia, distribuye. Los Nodes reciben datos, los procesan y los reenvían al siguiente componente.

03 / DESTINATION

Escribe datos

Guarda en CSV, carga en PostgreSQL, sube a BigQuery. Una Destination no tiene salidas — es el final del pipeline.

04 / PUMP

Ejecuta con un llamado

Una vez conectados los componentes, origin.pump() dispara todo el pipeline. Sin corchetes de orquestación ni DAGs.

pipeline.py 
from open_stage.core.base import Pipe
from open_stage.core.common import CSVOrigin, Filter, Aggregator, CSVDestination

# 1. Definir componentes
origin  = CSVOrigin("ventas", filepath_or_buffer="ventas.csv")
filtro  = Filter("solo_2024", field="año", condition="=", value_or_values=2024)
agrega  = Aggregator("por_region", key="region", agg_field_name="total",
                     agg_type="sum", field_to_agg="monto")
destino = CSVDestination("resultado", path_or_buf="ventas_2024.csv", index=False)

# 2. Conectar con pipes (method chaining)
origin.add_output_pipe(Pipe("p1")).set_destination(filtro)
filtro.add_output_pipe(Pipe("p2")).set_destination(agrega)
agrega.add_output_pipe(Pipe("p3")).set_destination(destino)

# 3. Ejecutar
origin.pump()

Ejemplos

Casos de uso reales

Desde una migración entre bases de datos hasta transformación con IA — todo con el mismo patrón.

Filtrar y agregar

CSV → Filter → Aggregator → CSV

from open_stage.core.base import Pipe
from open_stage.core.common import (
    CSVOrigin, Filter, Aggregator, CSVDestination
)

origin = CSVOrigin("ventas", filepath_or_buffer="ventas.csv")
filtro = Filter("alto_valor", field="monto",
               condition=">", value_or_values=1000)
agrega = Aggregator("por_region", key="region",
                   agg_field_name="total", agg_type="sum",
                   field_to_agg="monto")
destino = CSVDestination("resumen",
                         path_or_buf="resumen.csv", index=False)

origin.add_output_pipe(Pipe("p1")).set_destination(filtro)
filtro.add_output_pipe(Pipe("p2")).set_destination(agrega)
agrega.add_output_pipe(Pipe("p3")).set_destination(destino)
origin.pump()

Migración entre bases de datos

MySQL → PostgreSQL

from open_stage.core.base import Pipe
from open_stage.mysql.common import MySQLOrigin
from open_stage.postgres.common import PostgresDestination

origin = MySQLOrigin(
    name="fuente",
    host="localhost", database="origen",
    user="root", password="...",
    query="SELECT * FROM clientes WHERE activo = 1",
)
destino = PostgresDestination(
    name="destino",
    host="localhost", database="destino",
    user="postgres", password="...",
    table="clientes", if_exists="append",
)

origin.add_output_pipe(Pipe("migrar")).set_destination(destino)
origin.pump()

Bifurcación de flujo

Switcher distribuye, Funnel reúne

from open_stage.core.base import Pipe
from open_stage.core.common import (
    OpenOrigin, Switcher, Funnel, Printer
)
import pandas as pd

df = pd.DataFrame({
    "producto":  ["A", "B", "C", "D"],
    "categoria": ["elec", "ropa", "elec", "ropa"],
})

origin   = OpenOrigin("datos", df)
switcher = Switcher("por_cat", field="categoria",
                   mapping={"elec": "p_elec", "ropa": "p_ropa"})
funnel   = Funnel("reunir")
printer  = Printer("salida")

origin.add_output_pipe(Pipe("entrada")).set_destination(switcher)
switcher.add_output_pipe(Pipe("p_elec")).set_destination(funnel)
switcher.add_output_pipe(Pipe("p_ropa")).set_destination(funnel)
funnel.add_output_pipe(Pipe("salida")).set_destination(printer)
origin.pump()

Transformación con función propia

Cualquier lógica Python como Node

from open_stage.core.base import Pipe
from open_stage.core.common import (
    CSVOrigin, Transformer, CSVDestination
)

def aplicar_impuesto(df, tasa, envio):
    df = df.copy()
    df["precio_final"] = df["precio"] * (1 + tasa) + envio
    return df

origin = CSVOrigin("productos", filepath_or_buffer="productos.csv")
tx = Transformer(
    name="precio_final",
    transformer_function=aplicar_impuesto,
    transformer_kwargs={"tasa": 0.16, "envio": 50},
)
destino = CSVDestination("resultado",
                         path_or_buf="con_precio.csv", index=False)

origin.add_output_pipe(Pipe("p1")).set_destination(tx)
tx.add_output_pipe(Pipe("p2")).set_destination(destino)
origin.pump()

LLMs como Nodes en tu pipeline

ai_pipeline.py 
from open_stage.core.base import Pipe
from open_stage.core.common import CSVOrigin, CSVDestination
from open_stage.anthropic.claude import AnthropicPromptTransformer

origin = CSVOrigin("resenas", filepath_or_buffer="resenas.csv")

ai = AnthropicPromptTransformer(
    name="sentimiento",
    model="claude-sonnet-4-5-20250929",
    api_key="sk-ant-...",
    prompt="Agrega una columna 'sentimiento' con: positivo, neutro o negativo, "
           "basándote en la columna 'resena'.",
)

destino = CSVDestination("resultado", path_or_buf="clasificadas.csv", index=False)

origin.add_output_pipe(Pipe("p1")).set_destination(ai)
ai.add_output_pipe(Pipe("p2")).set_destination(destino)

origin.pump()

Flujo interno del AI Transformer:
1. Recibe el DataFrame  →  2. Lo serializa a CSV  →  3. Lo envía al LLM junto con el prompt  →  4. Parsea la respuesta CSV de vuelta a DataFrame  →  5. Lo reenvía al siguiente componente
